# Report Writing Prompts

## Executive Summaries

1. **One-Page Executive Summary**
   Write a one-page executive summary of this report/document: [paste content or describe]. Include key findings, implications, and recommended actions. Assume the reader has 2 minutes.

2. **Board-Level Summary**
   Summarize this information for a board presentation: [paste content]. Focus on strategic implications, financial impact, and decisions needed. Use metrics where possible.

3. **Situation-Complication-Resolution Summary**
   Restructure this information using SCR format: [paste content]. Start with the situation, present the complication, then lead to the resolution/recommendation.

4. **Bottom-Line-Up-Front Summary**
   Rewrite this report using BLUF format: [paste content]. Put the conclusion and recommendation first, then supporting details.

5. **Key Takeaways Extraction**
   Extract 3-5 key takeaways from this document: [paste content]. Make each actionable and memorable.

## Business Reports

6. **Quarterly Business Review Report**
   Create a QBR report structure for [department/business unit]. Include KPI performance, achievements, challenges, and forward-looking initiatives.

7. **Monthly Status Report**
   Generate a monthly status report template for [project/department]. Cover progress against goals, resource utilization, risks, and next month priorities.

8. **Annual Report Narrative**
   Write the narrative section of an annual report for [company/department]. Cover the year's journey, achievements, challenges overcome, and future outlook.

9. **Competitive Analysis Report**
   Structure a competitive analysis report on [competitor(s)]. Include market positioning, strengths/weaknesses, recent moves, and strategic implications.

10. **Market Research Report**
    Create a framework for a market research report on [market/topic]. Include methodology, key findings, market sizing, trends, and recommendations.

## Data & Analysis Reports

11. **Data Findings Report**
    Present these data findings in a clear report: [paste data or describe]. Include context, methodology, key insights, limitations, and recommended actions.

12. **KPI Dashboard Narrative**
    Write narrative commentary for these KPI results: [paste metrics]. Explain what the numbers mean, trends to note, and areas needing attention.

13. **Trend Analysis Report**
    Analyze these trends and write a report: [paste trend data]. Identify patterns, likely causes, implications, and what to monitor going forward.

14. **A/B Test Results Report**
    Report on A/B test results: [paste test details and results]. Cover hypothesis, methodology, statistical significance, findings, and recommendations.

15. **Survey Results Report**
    Transform these survey results into a report: [paste results]. Include methodology, key findings, notable patterns, and recommended actions.

## Technical & Operational Reports

16. **Technical Specification Document**
    Create a technical specification document for [feature/system]. Include requirements, architecture overview, dependencies, and acceptance criteria.

17. **Incident Report**
    Write an incident report for [describe incident]. Cover timeline, root cause, impact, response actions, and preventive measures.

18. **Process Documentation**
    Document this process: [describe process]. Include purpose, steps, responsible parties, exceptions, and success metrics.

19. **System Audit Report**
    Structure a system audit report for [system/process]. Include scope, methodology, findings (by severity), and recommendations.

20. **Capacity Planning Report**
    Create a capacity planning report for [resource/system]. Include current utilization, growth projections, constraints, and scaling recommendations.

## Financial Reports

21. **Financial Summary Report**
    Summarize this financial data for non-finance stakeholders: [paste data]. Explain key metrics in plain language with business context.

22. **Budget Variance Report**
    Create a budget variance report showing [actual vs budget data]. Explain significant variances, root causes, and corrective actions.

23. **ROI Analysis Report**
    Write an ROI analysis report for [initiative/investment]. Include methodology, assumptions, calculations, sensitivity analysis, and recommendation.

24. **Cost-Benefit Analysis**
    Create a cost-benefit analysis report for [proposal]. Structure costs, benefits (quantified where possible), risks, and net recommendation.

25. **Expense Report Summary**
    Summarize department expenses for [period]: [paste data]. Identify trends, notable items, and budget implications.

## Project Reports

26. **Project Proposal Document**
    Create a project proposal for [project idea]. Include business case, scope, approach, timeline, resources, risks, and success metrics.

27. **Project Status Report**
    Write a project status report for [project]. Cover overall health (RAG status), progress, accomplishments, risks, issues, and next steps.

28. **Project Closure Report**
    Create a project closure report for [completed project]. Include objectives vs outcomes, budget performance, lessons learned, and handoff items.

29. **Risk Assessment Report**
    Write a risk assessment report for [project/initiative]. Identify risks, assess likelihood and impact, document mitigations, and assign owners.

30. **Change Request Document**
    Document a change request for [change to project/process]. Include reason for change, impact analysis, alternatives considered, and approval needed.

## Strategic Reports

31. **SWOT Analysis Report**
    Create a SWOT analysis report for [company/product/initiative]. Include detailed findings for each quadrant and strategic implications.

32. **Strategic Recommendation Report**
    Write a strategic recommendation report for [decision/opportunity]. Include analysis, options evaluated, recommended approach, and implementation considerations.

33. **Business Case Document**
    Develop a business case for [proposed initiative]. Include problem statement, proposed solution, benefits, costs, risks, and implementation plan.

34. **Due Diligence Report**
    Structure a due diligence report for [acquisition/partnership/vendor]. Include areas of review, findings, red flags, and recommendation.

35. **Strategic Plan Summary**
    Summarize a strategic plan for [planning period]. Include vision, goals, key initiatives, success metrics, and resource requirements.

## HR & People Reports

36. **Employee Engagement Report**
    Create an employee engagement report from this data: [paste survey results]. Include key metrics, trends, areas of concern, and recommended actions.

37. **Headcount Report**
    Write a headcount report for [department/company]. Include current state, changes, open positions, and workforce planning considerations.

38. **Training Effectiveness Report**
    Report on training effectiveness for [program]. Include participation, feedback, knowledge retention, and behavior change metrics.

39. **Diversity & Inclusion Report**
    Create a D&I report covering [metrics/initiatives]. Present data sensitively, highlight progress, acknowledge gaps, and outline next steps.

40. **Turnover Analysis Report**
    Analyze turnover data and write a report: [paste data]. Include patterns, likely causes, impact, and retention recommendations.

## Communication-Focused Reports

41. **Stakeholder Update Report**
    Create a stakeholder update report for [initiative]. Tailor for audience understanding, cover what they care about, and manage expectations.

42. **Progress Report for Executives**
    Write an executive-friendly progress report on [initiative]. Focus on outcomes and decisions needed, not activities.

43. **Client Deliverable Report**
    Create a client-facing report on [project/analysis]. Balance comprehensiveness with readability, and make next steps clear.

44. **Board Deck Content**
    Write content for board deck slides on [topic]. Make each slide have one clear message with supporting data.

45. **Internal Newsletter Content**
    Write internal newsletter content on [topic/update]. Make it engaging, informative, and appropriately detailed for a broad employee audience.

## Report Enhancement

46. **Report Restructuring**
    Restructure this report for clarity: [paste report]. Improve logical flow, add section headers, and ensure key points stand out.

47. **Data Visualization Recommendations**
    Recommend charts/visualizations for this data: [paste data]. Specify chart type, what it should emphasize, and any design considerations.

48. **Report Editing for Conciseness**
    Edit this report to be 30% shorter while preserving key information: [paste report]. Cut redundancy and tighten language.

49. **Adding Context to Data**
    Add appropriate context to these findings: [paste data/findings]. Include comparisons, benchmarks, and "so what" interpretation.

50. **Executive Presentation Conversion**
    Convert this detailed report into a 10-slide executive presentation: [paste report]. Include key points, supporting data, and recommendations.
