# HTML Style Guide

## General Principles
*   Use semantic HTML elements (e.g., `<header>`, `<main>`, `<section>`, `<footer>`).
*   Always include `alt` attributes for images.
*   Keep the structure clean and well-indented.

## Attributes
*   Use double quotes for attribute values.
*   Order attributes logically (e.g., `class`, `id`, `name`, `data-*`, `src`, `href`).

## Formatting
*   Use 2 spaces for indentation.
*   Use lowercase for all tags and attributes.
*   Avoid trailing whitespace.
